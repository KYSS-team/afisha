# Frontend (Next.js)

Интерфейс электронной афиши построен на Next.js 14 с использованием TypeScript и UI-компонентов на базе Tailwind CSS. Разделы включают ленту событий, страницы аутентификации и административные экраны.

## Подготовка
1. Установите Node.js 18+ и npm.
2. Укажите адрес backend через переменную окружения `NEXT_PUBLIC_API_URL` (по умолчанию используется `http://localhost:8080`, задано в `app/auth/utils.ts`).
3. Установите зависимости:
   ```bash
   cd frontend
   npm install
   ```

## Локальный запуск
```bash
npm run dev
```
Приложение поднимется на `http://localhost:3000`.

## Основные экраны
- Главная страница приветствует пользователя и ведёт к каталогу событий и регистрации.
- `/events` — вкладки «Мои», «Активные» и «Прошедшие» события с карточками, изображениями и быстрыми действиями (подтвердить/отменить участие, перейти к редактированию).
- `/auth/*` — регистрация, вход, восстановление доступа; проверка форм реализована на клиенте.
- `/admin` — навигация в административную часть для модерации пользователей и событий.

Для загрузки картинок карточек используется endpoint backend `/events/{id}/image`, а API-вызовы выполняются через общий axios-инстанс `api` c базовым URL `API_BASE_URL`.
